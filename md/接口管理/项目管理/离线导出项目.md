## 离线导出项目
eoLinker AMS中提供了将项目文件离线导出的功能，各版本功能区别如下：

| | 基础免费版  | 企业专业版  | 企业私有云版  |
| ------------ | ------------ | ------------ | ------------ |
| eoLinker专用格式  | √  | √  |  √ |
| HTML网页  | √  | √  |√|
| PDF  |   | √  |√|
| Word  |   | √  |√|
| Markdown  |   | √  |√|

在项目概况页面中，点击 **导出项目** 按钮，选择需要导出的文件格式即可：
